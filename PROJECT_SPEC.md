# Abyss Breaker Project Spec

## 현재 버전

- 통합 버전: v3.0.0
- 장르: 실시간 막대 반사형 로그라이트 벽돌깨기
- 실행 방식: 외부 라이브러리, 외부 이미지, 아이콘 폰트, ES Module, npm, 빌드 도구 없이 정적 HTML/CSS/JavaScript 실행
- 저장 방식: localStorage `abyssBreaker.save.v6`
- 저장 스키마: `schemaVersion: 6`

## 핵심 게임 루프

1. 로비에서 게임 모드, 클래스, 영구 성장, 기록, 업적, 도감, 외형, 설정을 확인한다.
2. 런 시작 후 패들로 공을 반사해 벽돌, 기믹, 보스를 공략한다.
3. 각 스테이지에는 미션이 1개 배정되며, 성공 시 추가 점수와 심연석을 받는다.
4. 스테이지 클리어 후 능력을 선택하고, 보스 처치 후 유물을 선택한다.
5. 게임오버 또는 20스테이지 클리어 시 런 요약, 보상, 기록, 업적, 외형 해금을 정산한다.
6. 발견한 능력/유물/진화/아이템/보스/구역은 도감에 기록된다.

## v3.0.0 기준 기능

- 표준 모드: 1~20스테이지 선형 진행
- 보스 스테이지: 5, 10, 15, 20
- 보스 구성: 심연 감시자, 공허의 문지기, 거울 군주, 심연 핵
- 보스 방어 블록: 파괴 가능한 임시 방어 블록만 사용하며, 오래 공략 불가능한 상태가 되지 않도록 수량/시간/강제 약점 시간을 제한한다.
- 보스 약점 기믹: 보스별 약점 규칙과 표시를 제공한다.
- 스테이지 미션: 20개 스테이지에 미션을 배정하고 성공/실패/보상을 기록한다.
- 구역 규칙: 관문, 회랑, 균열로, 심장 구역마다 기믹과 아이템 가중치가 다르다.
- 도감: 발견한 능력, 유물, 진화, 아이템, 보스, 구역을 저장한다.
- 외형: 공/패들 스킨을 해금하고 선택 상태를 저장한다.
- 피드백: 효과음과 진동을 설정으로 켜고 끌 수 있다.
- 변형 모드: 반전 심연, 과부하 심연, 균열 심연을 고난도 도전 모드로 제공한다.
- 한글화: 사용자 표시 문구는 한국어를 기준으로 한다.
- UI 픽토그램: `icons.js`의 내장 SVG만 사용한다.

## 저장 안정성

- v1~v5 저장 데이터는 v6 기본 구조로 마이그레이션한다.
- 누락 필드는 기본값으로 보정한다.
- 클래스 해금 상태는 `unlockedClasses`와 `unlocks.classes`를 동기화한다.
- 클래스 비용은 균형자 0, 수호자 60, 파괴자 60, 연금술사 100, 조율자 120이다.
- 체크포인트에는 미션, 도감/외형 선택 참조, 런 통계, 구역/진화/유물 상태를 포함한다.

## 이번 버전에서 제외한 기능

- 심연탑
- 보스 러시
- 구역 변이
- 클래스 숙련도
- 시드 공유
- 로컬 랭킹 확장
- 컬렉션 룸
