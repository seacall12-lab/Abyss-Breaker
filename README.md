# Abyss Breaker

모바일 세로 화면에 맞춘 실시간 막대 반사형 벽돌깨기입니다. 현재 버전은 통합 v1.0.0이며, 별도 빌드 도구 없이 `index.html`을 브라우저에서 바로 실행합니다.

## 실행 방법

1. 저장소 폴더를 엽니다.
2. 브라우저에서 `index.html`을 실행합니다.
3. GitHub Pages에는 루트 폴더 그대로 배포하면 됩니다.

## 조작

- 모바일: 전투 영역을 좌우로 드래그해 패들을 이동합니다.
- PC: 전투 영역 위에서 마우스를 움직이거나 드래그합니다.
- 발사: `발사` 버튼 또는 준비 상태에서 전투 영역 탭/클릭.
- 일시정지: `일시정지` 버튼.
- 로비: 모드, 클래스, 영구 성장, 업적, 기록, 설정을 관리합니다.

## v1.0.0 주요 기능

- 표준, 무한, 단일 생명, 아이템 봉인, 고속 심연 게임 모드
- 표준 모드 첫 클리어 후 무한/도전 모드 해금
- 무한 모드 10스테이지 이후 난이도, 보스, 보상 스케일링
- 15개 업적과 심연석 보상
- 모드별 최고 점수, 최고 스테이지, 최단 클리어, 누적 기록
- 저장 스키마 v3, v1/v2 저장 데이터 마이그레이션
- 저장 내보내기, 불러오기, 전체 초기화
- 사운드, 진동, 효과 감소 설정
- v0.7의 클래스, 유물, 심연석 정산, 영구 성장 유지

## 파일 역할

- `index.html`: DOM 구조와 스크립트 로딩
- `style.css`: 모바일 세로 UI, HUD, 오버레이, 설정/기록 화면
- `data.js`: 상수, 스테이지, 능력, 보스, 클래스, 유물, 모드, 업적, 저장 기본값
- `state.js`: 런/영구 상태, localStorage 저장, 마이그레이션, 설정/저장 관리
- `game.js`: 게임 규칙, 물리, 충돌, 보상, 모드/도전/업적/기록 연결
- `render.js`: Canvas 렌더링
- `ui.js`: HUD, 로비, 오버레이, 버튼, Pointer Events, 저장 관리 UI
- `main.js`: 초기화, 리사이즈, 단일 게임 루프
