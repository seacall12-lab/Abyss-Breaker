# Abyss Breaker

Abyss Breaker는 모바일 세로 화면에 맞춘 벽돌깨기 로그라이크 프로토타입입니다. v0.1은 조준, 발사, 벽돌 충돌, 턴 진행, 플레이어 HP, 게임오버까지 한 화면에서 반복 플레이할 수 있는 코어 루프 검증 버전입니다.

## 실행 방법

설치나 빌드 과정은 없습니다.

1. 저장소 파일을 그대로 둡니다.
2. `index.html`을 브라우저에서 엽니다.
3. GitHub Pages에서는 루트의 `index.html`이 바로 실행됩니다.

## 조작법

- 시작: 시작 버튼을 누릅니다.
- 조준: Canvas 하단의 발사 위치에서 드래그합니다.
- 발사: 손가락이나 마우스를 놓습니다.
- 일시정지: 하단의 일시정지 버튼을 누릅니다.
- 재시작: 하단 또는 오버레이의 재시작 버튼을 누릅니다.

터치와 마우스는 Pointer Events로 처리합니다. 별도의 mouse/touch 이벤트는 사용하지 않습니다.

## v0.1 구현 기능

- 모바일 세로 한 화면 레이아웃
- Canvas 기반 전투 화면
- 드래그 조준과 순차 공 발사
- 좌우/상단 벽 반사
- 원-사각형 벽돌 충돌과 피해 계산
- 빠른 공 관통 방지를 위한 서브 스텝
- 벽돌 HP 표시, 피해 숫자, 파티클, 화면 흔들림
- 모든 공 복귀 후 턴 종료
- 살아남은 벽돌 하강과 위험선 도달 시 플레이어 피해
- 웨이브 진행과 게임오버
- 최고 웨이브 localStorage 저장
- 시작, 일시정지, 계속하기, 재시작 UI

## 파일 구조

- `index.html`: DOM 구조와 스크립트 로딩
- `style.css`: 모바일 한 화면 레이아웃과 UI 스타일
- `data.js`: 상수와 밸런스 데이터
- `state.js`: 런 상태와 저장 데이터
- `game.js`: 조준, 발사, 물리, 충돌, 턴 처리
- `render.js`: Canvas 렌더링
- `ui.js`: DOM, HUD, 오버레이, 입력 이벤트
- `main.js`: 초기화, 리사이즈, 게임 루프
