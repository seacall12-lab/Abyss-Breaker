# Abyss Breaker

모바일 세로 화면에 맞춘 실시간 패들 반사형 벽돌깨기 v0.1입니다. 빌드 도구 없이 `index.html`을 브라우저에서 바로 열어 실행합니다.

## 실행 방법

1. 이 폴더의 파일을 그대로 둡니다.
2. 브라우저에서 `index.html`을 엽니다.
3. GitHub Pages에는 루트 폴더 그대로 배포하면 됩니다.

## 조작

- 모바일: Canvas 위에서 좌우로 드래그해 패들을 이동합니다.
- PC: Canvas 위에서 마우스를 움직이거나 드래그해 패들을 이동합니다.
- 발사: `발사` 버튼 또는 준비 상태에서 Canvas 클릭/탭.
- 일시정지: `일시정지` 버튼.
- 재시작: `재시작` 버튼.

## v0.1 기능

- 실시간 패들 반사형 벽돌깨기
- 생명 3개, 점수, 최고 점수 저장
- 벽/패들/벽돌 충돌과 반사각 처리
- 벽돌 HP와 파괴 보상
- 스테이지 클리어, 생명 손실, 게임오버 오버레이
- 아이템 3종: 패들 확장, 멀티볼, 공 감속
- 파티클, 점수 텍스트, 간단한 화면 흔들림
- 모바일 세로 레이아웃, 스크롤 차단, Pointer Events 입력

## 파일 역할

- `index.html`: DOM 구조와 스크립트 로딩
- `style.css`: 모바일 세로 UI와 오버레이 스타일
- `data.js`: 상수, 밸런스, 아이템, 스테이지 패턴
- `state.js`: 런타임 상태와 localStorage 저장
- `game.js`: 패들, 공, 벽돌, 아이템, 충돌, 모드 전환
- `render.js`: Canvas 렌더링
- `ui.js`: HUD, 오버레이, 버튼, Pointer Events
- `main.js`: 초기화, 리사이즈, 게임 루프
