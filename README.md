# Abyss Breaker

모바일 세로 화면에 맞춘 실시간 반사형 로그라이트 벽돌깨기 게임입니다. 현재 버전은 v5.1.0이며, 외부 라이브러리, 외부 이미지, 아이콘 폰트, ES Module, npm, 빌드 도구 없이 정적 HTML/CSS/JavaScript로 실행합니다.

## 실행 방법

1. 저장소 폴더를 엽니다.
2. 브라우저에서 `index.html`을 실행합니다.
3. GitHub Pages에는 루트 폴더 그대로 배포하면 됩니다.

## 조작

- 모바일: 전투 영역을 좌우로 드래그해 패들을 이동합니다.
- PC: 전투 영역 위에서 마우스를 움직이거나 드래그합니다.
- 발사: `발사` 버튼 또는 준비 상태에서 전투 영역 클릭.
- 일시정지: `일시정지` 버튼.
- 로비: 게임 모드, 클래스, 영구 성장, 장착, 심연 연구소, 업적, 기록, 도감, 외형, 설정을 관리합니다.

## 파일 역할

- `index.html`: DOM 구조와 스크립트 로딩
- `style.css`: 모바일 세로 UI, HUD, 오버레이, 카드, 픽토그램 스타일
- `data.js`: 버전, 상수, 20스테이지, 미션, 보스, 능력, 클래스, 유물, 모드, 도감/외형 기본값
- `state.js`: 런/영구 상태, localStorage 저장, 마이그레이션, 설정/도감/외형 관리
- `game.js`: 게임 규칙, 물리, 충돌, 미션, 보스 약점, 보상, 모드/도전/기록 연결
- `render.js`: Canvas 렌더링
- `icons.js`: 외부 자산 없는 내장 SVG 픽토그램
- `ui.js`: HUD, 로비, 오버레이, 버튼, 카드, Pointer Events, 저장 관리 UI
- `main.js`: 초기화, 리사이즈, 피드백, 단일 게임 루프

## 알려진 제한

- 저장 데이터는 브라우저 localStorage에만 보관됩니다.
- 사운드는 Web Audio API로 간단한 효과음만 생성합니다.
- 서비스 워커 캐시는 보안 컨텍스트에서만 등록됩니다.
