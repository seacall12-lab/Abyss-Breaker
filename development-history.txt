개발 기록

v0.4.0
- 단일 실시간 패들 반사 플레이를 1~10 스테이지 런 구조로 확장했다.
- 스테이지 클리어마다 런 능력을 선택하는 흐름을 추가했다.
- 강화, 벽, 폭발, 아이템 벽돌을 추가했다.
- 5스테이지와 10스테이지 보스를 추가했다.
- 보스 보상, 런 클리어, 최고 기록 저장을 추가했다.

v0.7.0
- 로비, 클래스 선택/해금, 영구 성장 UI를 추가했다.
- 5스테이지 보스 클리어 후 유물 선택 흐름을 추가했다.
- 게임오버와 런 클리어에서 심연석을 1회만 정산하도록 `runRewardGranted` 플래그를 추가했다.
- 클래스, 영구 성장, 유물 효과를 패들 폭, 생명, 공 속도, 벽돌/보스 피해, 점수, 아이템 드롭 계산에 연결했다.
- localStorage 저장 스키마를 v2로 확장하고 v1 저장값 마이그레이션을 추가했다.

v1.0.0
- 게임 모드 선택을 추가하고 표준 모드 클리어 후 무한/도전 모드가 해금되도록 연결했다.
- 무한 모드는 10스테이지 이후 반복 패턴, 보스 주기, 유물 최대 3개, 난이도/보상 스케일링을 적용한다.
- 단일 생명, 아이템 봉인, 고속 심연 도전 모드를 구현하고 각 도전 규칙을 능력/아이템/생명 계산에 연결했다.
- 업적과 보상을 추가하고 중복 지급을 저장 상태로 차단했다.
- 표준, 무한, 도전, 클래스별 기록과 누적 기록을 저장한다.
- 저장 스키마를 v3로 확장하고 v1/v2 저장값을 v3 기본 구조로 마이그레이션한다.
- 저장 내보내기, 불러오기, 전체 초기화와 설정 UI를 추가했다.
- 효과 감소 설정을 렌더링에 반영하고 모바일 오버레이 스크롤 안정성을 보강했다.
v1.1.0
- 저장 스키마를 v4로 올리고 `activeRun` 체크포인트를 추가했다.
- 로비에서 이어하기/새로 시작 UI를 표시하도록 연결했다.
- 패들 하단 여백, Canvas DPR 상한, 공 반지름과 외곽선 두께를 조정했다.
- 벽돌 충돌 쿨다운 처리와 HUD/UI DOM 갱신을 안정화했다.
v1.5.0
- 구역과 스테이지 기믹 보조 데이터 구조를 추가했다.
- 범퍼, 포털, 이동 거울, 회전 반사자, 보호막 벽돌 충돌 처리를 추가했다.
- 능력·유물 조합 기반 진화 6종을 이벤트 기반으로 발동하도록 연결했다.
- 자석 막대, 레이저 막대, 하단 보호막 아이템 상태 관리를 추가했다.
- 보스 HP 페이즈에 따라 기믹을 추가하고 무한 모드 구역 순환을 적용했다.
- 체크포인트에 진화와 구역 정보를 포함하되 실시간 기믹/아이템 위치는 저장하지 않도록 유지했다.
- 모바일 성능을 위해 기믹, 레이저, 폭발 연쇄 개수 상한을 데이터로 제한했다.

v2.1.0
- 보스 방어 블록이 공략 불가능 상태를 만들던 문제를 수정했다.
- 파괴 불가 블록의 보스전 방어 용도 생성을 차단했다.
- 보스가 오래 피해를 받지 않으면 약점이 강제로 열리는 타이머를 추가했다.
- 표준 모드를 20스테이지로 확장하고 5, 10, 15, 20스테이지 보스 구조로 정리했다.
- 사용자 표시 문구를 한국어 기준으로 정리했다.
- `icons.js`에 실제 SVG 픽토그램을 추가하고 로비, 카드, HUD, 기록, 설정 UI에 적용했다.

v2.2.0
- 클래스 해금이 안 되거나 불명확하게 보이던 문제를 안정화했다.
- `unlockedClasses`와 `unlocks.classes`를 로드와 저장 시 양방향 동기화하도록 정리했다.
- 클래스 비용 최종 기준을 균형자 0, 수호자 60, 파괴자 60, 연금술사 100, 조율자 120으로 통일했다.
- 잠긴 클래스 카드에 현재 심연석과 필요 비용, 부족 상태를 표시하도록 개선했다.
- 사용자 표시 영문 문구와 진화 명칭을 한국어 기준으로 정리했다.
- 10스테이지와 20스테이지 보스가 방어 블록 때문에 공략 불가능해지지 않는 규칙을 재점검했다.
- v3 준비용으로 능력, 아이템, 클래스의 `iconId`, `tags`, `category` 일부를 정리했다.

v3.0.0
- 20개 스테이지에 미션을 배정하고 성공/실패/보상 정산을 런 진행과 결과 화면에 연결했다.
- 관문, 회랑, 균열로, 심장 구역별 전용 규칙과 아이템/기믹 가중치를 정리했다.
- 5, 10, 15, 20스테이지 보스에 약점 규칙과 약점 표시를 추가했다.
- 런 결과 요약, 도감 발견 기록, 공/패들 스킨 해금과 선택 저장을 구현했다.
- Web Audio 기반 효과음과 `navigator.vibrate` 기반 진동 피드백을 설정과 연결했다.
- 반전 심연, 과부하 심연, 균열 심연 변형 모드를 추가하고 도전 기록 대상으로 분류했다.
- 저장 스키마를 v6으로 올리고 v1~v5 저장 데이터와 체크포인트를 새 구조로 보정하도록 확장했다.
- 심연탑, 보스 러시, 구역 변이, 클래스 숙련도, 시드 공유, 로컬 랭킹 확장, 컬렉션 룸은 v4/v5 범위로 남겼다.

v4.0.0
- 심연탑은 기존 스테이지 생성기를 재사용하되 층 단위 스케일링, 5층 보스, 10층 강화 보스, 층별 변이를 적용하도록 분리했다.
- 보스 러시는 5연전 보스 전용 모드로 추가하고 일반 진행 보상 대신 보스 사이 능력 선택 흐름을 사용하도록 구성했다.
- 구역 변이는 데이터 기반 `MUTATIONS`로 정의하고 표준/무한/심연탑 스테이지 배율, 이동 벽돌, 보상 배율에 반영했다.
- 클래스 숙련도는 클래스별 경험치와 1~10레벨 구조로 저장하고 런 정산 시 보스, 미션, 클리어 성과를 경험치로 환산했다.
- 빌드 아키타입은 능력, 유물, 아이템, 약점, 포털, 레이저, 방어 이벤트를 점수화해 대표/보조 빌드를 기록하도록 구현했다.
- 엔드게임 보상은 신규 공/패들 스킨과 엠블럼으로 제한해 밸런스 영향 없이 장기 목표를 제공하도록 설계했다.
- 저장 스키마를 v7로 올리고 v6 이하 저장 데이터와 체크포인트가 새 엔드게임 필드를 기본값으로 보정하도록 확장했다.
- 시드 공유, 런 결과 공유, 로컬 랭킹 대형 확장, 컬렉션 룸, 접근성 최종 정리, 도움말 대형 개편은 v5 범위로 유지했다.

v5.0.0
- 공 코어, 보드 프레임, 스킬 칩을 데이터 기반 장착 아이템으로 분리했다.
- 추출 결과는 신규 보유 또는 중복 심연 조각으로 처리하고 희귀/영웅/전설 천장 카운터를 저장한다.
- 심연 조각 선택 제작 구조를 추가하고 보유 여부와 조각 부족 상태를 저장 단계에서 보정한다.
- 장착 프리셋 3개를 저장하고 런 시작 시 장착과 접근성 설정을 체크포인트 스냅샷으로 고정한다.
- 공격형, 생존형, 아이템형, 정밀형 추천 장착은 보유 아이템 태그와 등급 점수로 자동 선택한다.
- 도감에 공 코어, 보드 프레임, 스킬 칩 컬렉션 탭을 추가했다.
- 접근성 설정에 공 크기, 막대 크기, 터치 민감도, 고대비, 효과 감소, 화면 흔들림을 추가했다.
- 저장 스키마를 v8로 올리고 v7 이하 저장 데이터를 v8 기본 구조로 마이그레이션한다.
